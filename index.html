Internet Speed Test Automation Plan

An interactive overview of our plan to automate network performance monitoring using UiPath, providing reliable, real-time insights.

Executive Summary

This plan outlines the implementation of an automated agent using the UiPath platform to monitor internet speed from multiple locations. The agent will run on a scheduled basis, collect data, and automatically save it to a central Google Sheet. This will provide a consistent, reliable, and real-time view of our network performance without manual intervention, enabling proactive management and troubleshooting.

Phase 1: Build the Automation

Developing the Robot in UiPath Studio

The core automation is built using a visual, low-code interface. This creates a reusable process that performs the speed test. Click each step below to see the details.

1
Open Speed Test Site

The automation will open a web browser and navigate to a reliable internet speed test website.

2
Run the Test

It will use UI automation to interact with the webpage, clicking the "Go" or "Start" button to initiate the test.

3
Extract Data

Once the test is complete, the robot will use data scraping techniques to capture the key metrics: download and upload speed.

4
Log Data to Google Sheets

The extracted speed data, along with a timestamp and the specific location ID, will be written to a designated Google Sheet.

Phase 2: Deploy and Run the Agent

Centralized Management with UiPath Orchestrator

The automation is managed and scheduled centrally using Orchestrator, which acts as the control center for all robots, ensuring reliability and scalability.
