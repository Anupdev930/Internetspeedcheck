Internet Speed Test Automation Plan

An interactive overview of our plan to automate network performance monitoring using UiPath, providing reliable, real-time insights.

Executive Summary

This plan outlines the implementation of an automated agent using the UiPath platform to monitor internet speed and data usage from multiple locations. The agent will run on a scheduled basis, collect data, and automatically save it to a central Google Sheet. This will provide a consistent, reliable, and real-time view of our network performance without manual intervention, enabling proactive management and troubleshooting.

Phase 1: Build the Automation

Developing the Robot in UiPath Studio

The core automation is built using a visual, low-code interface. This creates a reusable process that performs the speed test and checks data usage. Click each step below to see the details.

1
Open Speed Test Site

The automation will open a web browser and navigate to a reliable internet speed test website.

2
Run the Speed Test

It will use UI automation to interact with the webpage, clicking the "Go" or "Start" button to initiate the test.

3
Securely Log In to Provider Portal

The robot will securely retrieve login credentials from UiPath Orchestrator and use UI automation to log into the internet provider portal (e.g., Airtel, Jio).

4
Extract Data Usage

It will navigate to the data usage section of the portal and scrape key metrics like available data limit and current usage.

5
Log Data to Google Sheets

All extracted data will be logged with a timestamp and location ID into a designated Google Sheet for centralized analysis.

Phase 2: Deploy and Run the Agent

Centralized Management with UiPath Orchestrator

The automation is managed and scheduled centrally using Orchestrator, which acts as the control center for all robots, ensuring reliability and scalability.
